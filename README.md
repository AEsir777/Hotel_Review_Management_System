# Hotel_Review_Management_System
## Database Drive Application
Node.JS

```
npm install
```
set up ```.env``` file
```
node index.js
```

## START front end
```
cd client-side
npm run dev
```


## server-side: REST API
```
cd server-side
nodemon(or node) index.js
```

GET ```/api/review/:rid```  
PUT ```/api/review/:rid```  
DELETE ```/api/review/:rid```  
POST ```/api/review/:rid/cool```
